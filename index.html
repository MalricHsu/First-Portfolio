    序言

    這個作品集展示了我在學習程式過程中完成的多個專案，涵蓋網頁設計、前端互動與 API 串接，使用了 HTML、CSS、JavaScript 及 React 等技術。透過這些專案，你可以看到我不斷學習、嘗試並成長的過程。

    • 圍棋資訊網使用 HTML和CSS打造，版面簡潔、結構清晰。
    • 部落格排版 完成響應式設計（RWD），提供多裝置友善的瀏覽體驗。
    • Todo List 一個 JavaScript 製作，另一個使用 React.js，練習不同技術架構。
    • 即時天氣 使用 JavaScript 串接天氣資料，實現即時天氣查詢功能。
    • 簡易電商平台包含商品展示與購物車功能，利用 JavaScript 實現互動邏輯。
    • React圖片庫 串接圖片 API，動態顯示多張圖片，練習 React 與 API 整合。

    程式之路

    結業證書

    我於 2024 年 11 月加入一家新創電商公司，擔任產品實習 PM，主要負責協助處理客服回報的產品功能異常與系統問題。這些問題經常涉及網站前後端的 Bug，為了提升問題判斷與跨部門溝通效率，我開始在下班後自學網頁開發相關技能，期望能在產品與工程團隊之間扮演更積極的橋樑角色。


    為了系統化學習，我選擇修習 Udemy 的《2025 網頁全端開發》課程，利用平日晚上與週末時間，在四個月內完成課程內容，並實作了九個教學專案，進一步鞏固了 HTML、CSS、JavaScript 等基礎技術的應用能力。

    目前，我正在參加由六角學院舉辦的《2025 軟體工程師體驗營》，這是一個為期 30 天的密集培訓營，內容聚焦於網頁排版技巧，學習 Bootstrap 5 的實務應用與響應式網頁設計（RWD）方法。接下來也將學習如何架設靜態網站，補足從前端切版到網站部署的完整流程。

    未來，我將持續在業餘時間精進程式開發技能，特別是加強對 React 框架的理解與實作經驗，期望朝全方位產品與前端開發能力發展，成為具備技術背景的產品人才，能夠有效橋接技術與商務需求，推動產品持續優化。

    作品介紹

    主要程式語言：HTML（91.3%) 、 CSS（8.7％）

    學習內容及心得：

    此作品為 Udemy 課程中的第一個練習作業，主要目的是了解網頁的基本架構，學習如何使用 HTML5 語法建立頁面，並將五個獨立的 HTML頁面串接在一起，完成一個具有基本導覽功能的靜態網站。 ⇒ 有引入JavaScript 動態背景特效插件canvas-nest.js

    遇到的困難

    在將 HTML 結構搭配 CSS 樣式時，常常會不知道該如何設計出理想的畫面呈現，透過一次又一次的練習與模仿，逐漸掌握排版的基本邏輯。
